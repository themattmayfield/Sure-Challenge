## Run Locally

Clone the project

```bash
  git clone https://github.com/themattmayfield/Sure-Challenge.git
```

Go to the project directory

```bash
  cd Sure-Challenge
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
